Welcome to Your Lean Six Sigma Toolkit

This application provides tools to support your process improvement projects following the DMAIC methodology.

Getting Started: Follow the DMAIC phases using the navigation above.

  1. Define: Start a new project with a Project Charter and SIPOC diagram.
  2. Measure: Plan data collection, import data (CSV/Excel), and consider Measurement System Analysis.
  3. Analyze: Calculate statistics, view control charts, and assess process capability using imported data. Explore root causes with tools like the Fishbone diagram.
  4. Improve: Explore potential improvements using predictive tools (if available) or brainstorming templates like FMEA.
  5. Control: Document your control strategy with a Control Plan and plan for ongoing monitoring.

Use the Graphical Forms tool for process maps and other diagrams, and the Capability Calc for quick standalone calculations.
